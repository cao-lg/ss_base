# 数智电商数据分析教学平台

基于 Coze 智能体的现代化教学网站，覆盖电商数据分析六大项目。

## 项目结构

```
├── index.html        # 主页
├── courses.html      # 课程列表
├── agents.html       # AI智能体中心
├── dashboard.html    # 学习画像仪表板
├── project1.html     # 项目一：开启数据驱动之旅
├── project2.html     # 项目二：商品选品数据分析
├── project3.html     # 项目三：营销推广数据分析
├── project4.html     # 项目四：销售交易数据分析
├── project5.html     # 项目五：客户服务数据分析
├── project6.html     # 项目六：物流履约数据分析
├── styles/main.css   # 样式
└── scripts/app.js    # 交互逻辑
```

## 部署

静态站点，无需构建。部署到 Cloudflare Pages 时：
- 构建命令：留空
- 输出目录：`/`
